# Zobo CLI

A command line utility with LLM capabilities.

## Installation

```bash
npm install -g zobo
```

Or run directly with npx:

```bash
npx zobo
```

## Commands

### hi

Get a greeting from the LLM:

```bash
zobo hi
```

## Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the project:
   ```bash
   npm run build
   ```
4. Run tests:
   ```bash
   npm test
   ```
   
To run integration tests:
```bash
npm run test:integration
```
