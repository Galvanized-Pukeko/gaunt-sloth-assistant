# v1.1.1 AI Ignore and Console Levels

## New Features
- **.aiignore Support:** Filesystem tools now respect `.aiignore` patterns. Configure via `aiignore.enabled` or provide patterns directly with `aiignore.patterns`.
- **Console Level Filtering:** Added `consoleLevel` to control which status messages appear in output. JSON configs accept string levels (e.g., `info`, `warning`, `debug`).

## Improvements
- Missing `.aiignore` files are reported at debug level only.
