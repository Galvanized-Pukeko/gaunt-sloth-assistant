# Development Guidelines - JS Config Example

This is an example project demonstrating JavaScript configuration with custom middleware and tools.

## Project Context

This example shows how to:
- Define custom middleware that logs lifecycle events
- Create custom tools using LangChain's tool API
- Configure Gaunt Sloth programmatically using JavaScript

## When Assisting

- Explain how middleware and tools work
- Show how to extend or modify the examples
- Demonstrate best practices for custom extensions
- Help troubleshoot configuration issues

## Code Style

- Use clear, descriptive names
- Add JSDoc comments for documentation
- Keep examples simple and focused
- Use console.log for debugging (this is example code)
